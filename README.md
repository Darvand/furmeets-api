# FurMeets API

## Description
Currently [FurMeets MiniApp](t.me/furmeets_bot/hub) use an API for some simple interactions:
- Telegram Bot: Allowing the service to requests information about users, group or send messages.
- Persistance: Some interactions with the MiniApp are stored to allow better interactions

## Tech Stack
- The backend is build using NestJS with TypeScript
- TelegramBot using [Grammy](https://www.npmjs.com/package/grammy)
- MongoDB as the database

## Methodology

Build following the code standards given by *Domain-Driven Design*

## Infraestructure

For a free deployment infraestructure, this repository uses [Render](render.com)
